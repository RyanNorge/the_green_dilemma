
## Installation

To run this project, you need to have UV installed. You can do this by running...

### On macOS and Linux.
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```
### On Windows.
```
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

It is also in Homebrew and probably most package mangagers. 

Or you can get more information about UV directly from their Github...

https://github.com/astral-sh/uv

## Running the Code
From the working directory, you just need to run 
```
python main.py
```
If you have UV installed then it will build a virtual enviroment and install packages for you.
